Tutorial rápido: React no PC e LifeOS

Este guia é para iniciantes. Para gerar um PDF: Ctrl+P (ou Cmd+P) → Salvar como PDF.

1. Instalar Node.js (inclui npm)

Baixe o instalador LTS em https://nodejs.org/, execute e marque a opção de adicionar ao PATH. Feche e reabra o terminal.

Confira:

node -v
npm -v

2. Abrir o projeto no VS Code

Menu Ficheiro → Abrir pasta e escolha LifeOSV7 (ou a pasta onde extraiu o projeto).

3. Instalar dependências do React (shell LifeOS)

cd frontend\react-app
npm install

4. Rodar o React em desenvolvimento

Com o Flask em http://127.0.0.1:5000 (porta padrão do LifeOS):

npm run dev

O Vite abrirá algo como http://localhost:5173. O ficheiro vite.config.ts já faz proxy de /api para o backend.

5. Dependências usadas

  • react e react-dom — interface
  • vite — servidor e empacotador de desenvolvimento
  • typescript — tipos (opcional mas recomendado)

6. Ligar React ao backend

O token Firebase fica em localStorage como lifeos_token depois do login no index.html.

No shell React, pode colar esse token na página de teste ou definir VITE_API_URL se o API estiver noutro domínio.

Pedidos autenticados usam o cabeçalho: Authorization: Bearer <token>.

7. Testar a dashboard

  1. Inicie o Flask (python main.py na pasta backend).
  2. Abra frontend/index.html (com servidor HTTP local) e faça login + onboarding.
  3. Confirme que a dashboard carrega (não fica em “Iniciando dashboard…”).
  4. Opcional: no React shell, cole o token e verifique se a prévia mostra nome/profissão.

8. Como saber se “Iniciando dashboard…” foi resolvido

  • O texto muda para “Carregando seus dados…” e o overlay some.
  • O nome aparece no canto superior direito e na saudação “Olá, …”.
  • Em caso de erro de rede, aparece mensagem de erro ou retry — não um loading eterno na primeira mensagem.
